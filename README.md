# Adventure JAVA 2D
Java game developed to enhance programming skills

## Primera parte

Realizamos el entendimiento de como se debe iniciar un JFrame con un JPanel el cual tendrá un objeto de Gráficos 2D para dibujar obejetos

## Segunda parte

Profundazamos en la parte del game loop usando el metodo sleep y el metodo delta, adicionalmente, creamos el KeyInput para mover el jugador.

## Tercera parte
Realizamos la inclusión sprites y animaciones al personaje.
